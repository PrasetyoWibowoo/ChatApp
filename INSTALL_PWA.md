## 📲 iPhone / iPad

### Langkah-langkah Install:

1. **Buka Safari** (harus pakai Safari, bukan Chrome!)
   - Kunjungi: https://chat-app-sigma-topaz-55.vercel.app

2. **Tap tombol Share** 
   - Cari icon kotak dengan panah ke atas di bagian bawah layar
   - Atau icon berbagi di address bar

3. **Pilih "Add to Home Screen"**
   - Scroll ke bawah sampai ketemu opsi ini
   - Biasanya ada icon plus (+) hijau

4. **Kasih nama** (opsional)
   - Default: "CollabNotes"
   - Bisa diganti sesuai keinginan

5. **Tap "Add"**
   - Icon akan muncul di home screen
   - Aplikasi siap dipakai!

### Cara Pakai:
- Tap icon di home screen
- Aplikasi buka fullscreen (tanpa Safari UI)
- Video call & notifikasi bekerja normal
- **Requirement: iOS 16.4+ untuk push notifications**

## 🐛 Troubleshooting

### ❌ Tidak bisa install di iPhone?
- **Solusi:** Harus pakai Safari, bukan Chrome/Firefox
- Chrome & Firefox di iOS tidak support PWA install

### ❌ Notifikasi tidak muncul?
- **iOS:** Cek iOS versi 16.4 atau lebih baru
- **Android:** Cek permission di Settings → Apps
- **PC:** Cek notification permission di browser settings

### ❌ Video call tidak jalan?
- **Cek:** Permission camera & microphone
- **iOS:** Settings → Safari → Camera/Microphone → Allow
- **Android:** Settings → Apps → [App Name] → Permissions
- **PC:** Browser settings → Privacy → Camera/Microphone

### ❌ Icon tidak muncul?
- **Reload website** dan coba install lagi
- **Clear browser cache**
- **Update browser** ke versi terbaru

### ❌ App lemot?
- **Clear app cache**: Uninstall → Reinstall
- **Check internet connection**
- **Update app**: Akan auto-update saat buka
